# Endgame with llama

An End game app built with React, Vite and Yarn.

Play: Guess the word and in limited attempts.

## Features

- As you hit wrong keys the languages fades.
- If you have reached till Assemly language without guessing the word, then you lost.
- Unselected key: Orange
- Correct Key: Green
- InCorrect Key: Red

## Prerequisites

- llama3.2 (ollama.com)
- Node.js (LTS recommended)
- Yarn (project uses Yarn)

## Install Llama
- Go to (ollama.com) to download Ollama. Run the downloaded file to begin the installation and follow the on-screen prompts
- verify

$ ollama -v

$ ollama serve

http://127.0.0.1:11434/
Ollama is running


## Install REACT dependencies:
yarn install


## Start the dev server and open the app:
yarn dev

http://localhost:5173

## Build a production bundle:

yarn build

## Gameplay / Controls
Click the keypad and guess the word!
