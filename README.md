An End game app built with React, Vite and Yarn.

Play: Guess the word and in limited attempts.

## Features

- As you hit wrong keys the languages fades.
- If you have reached till Assemly language without guessing the word, then you lost.
- Unselected key: Orange
- Correct Key: Green
- InCorrect Key: Red

## Prerequisites

- Node.js (LTS recommended)
- Yarn (project uses Yarn)


## Install dependencies:
yarn install


## Start the dev server and open the app:
yarn dev

http://localhost:5173

## Build a production bundle:

yarn build

## Gameplay / Controls
Click the keypad and go!
